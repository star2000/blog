有时, 我们想运行一个在线脚本, 又不想下载它
<!--more-->

## Windows
使用PowerShell执行  
如果脚本里包含非ASCII字符, 用

```ps1
$OutputEncoding=[Text.Encoding]::UTF8;(New-Object Net.WebClient -Property @{Encoding=[Text.Encoding]::UTF8}).DownloadString('https://blog.star2000.work/下载/代码/你好.py') | py
```

否则用

```ps1
(New-Object Net.WebClient).DownloadString('https://blog.star2000.work/下载/代码/hello.py') | py
```

## Linux

```sh
wget -qO- blog.star2000.work/下载/代码/你好.py | python3
```

或者

```sh
curl -sL blog.star2000.work/下载/代码/你好.py | python3
```
