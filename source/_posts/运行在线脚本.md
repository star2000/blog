有时，我们想运行一个在线脚本，又不想下载它

<!--more-->

## Windows

使用 PowerShell 执行  
如果脚本里包含非 ASCII 字符，比如汉字，先修改管道字符集

```powershell
$OutputEncoding=[Text.Encoding]::UTF8
```

再执行

```powershell
(iwr -useb $url).content | py
```

## Linux

```bash
wget -qO- $url | python3
```

或者

```bash
curl -sL $url | python3
```
